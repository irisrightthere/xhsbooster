# xhsbooster 网站设计规范

> 本次修改后的 UI 标准，后续开发按此执行。

## 页面结构

```
┌─────────────────────────────────────┐
│          🍠 xhsbooster              │
│       2026-06-21 · 韩娱资讯          │
├─────────────────────────────────────┤
│ [Soompi 30] [Google News 30]        │  ← 源胶囊 Tab
│ [AsianWiki 30]                      │
├─────────────────────────────────────┤
│                                     │
│ ┌───────────────────────────────┐  │
│ │ 标题（链接原文）           NEW │  │
│ │ Soompi · 0621soompi           │  │
│ │ ──────────────────────────── │  │
│ │ 正文摘要……                     │  │
│ └───────────────────────────────┘  │
│                                     │
├─────────────────────────────────────┤
│ xhsbooster · 韩娱资讯 · 历史归档    │  ← 页脚
└─────────────────────────────────────┘
```

## 卡片内容（精简版）

```
[标题]                    ← h3, 链接到原文
源名称 · MMDDsource       ← meta 行
NEW 标记                   ← 仅本轮新文章，闪烁红色
─────────────────         ← hr 分隔线
正文摘要                   ← summary_zh
```

## 不展示的内容

- ❌ 风险标记（低/中/高）
- ❌ 事实提取点（facts_list）
- ❌ 分类标签（韩娱/韩剧/日剧）
- ❌ 缩略图

## 归档页

- DailyBrief 风格：纯 `<ul>` 列表，日期链接，最新在前
- 入口：页面底部页脚 `历史归档` 链接
- 不分页

## Tab 规则

- 单层胶囊 Tab
- 排序：Soompi → AsianWiki → Google News → Google J-drama → Oricon
- 默认选中第一个
- 带文章计数
